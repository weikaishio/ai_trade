## 超短线选股的核心逻辑

**不是"选好股"，是"选最强的股"。** 超短线不关心公司好不好，只关心资金认不认。

---

## 完整流程分四个阶段

### 一、盘前准备（8:30-9:15）

**1. 判断市场情绪周期**

这是最重要的一步，决定今天该不该出手：

| 情绪阶段 | 特征 | 操作 |
|---------|------|------|
| 冰点 | 昨日跌停>涨停，连板股集体断板 | 轻仓试错龙头反包 |
| 回暖 | 有股票敢于反包，烂板开始修复 | 半仓做确定性最高的 |
| 升温 | 连板高度打开（3板+），涨停家数>50 | 正常仓位 |
| 高潮 | 涨停家数>100，遍地涨停 | **当天买入的大概率是最后一棒** |
| 退潮 | 昨日高潮，今日大面积闷杀 | **空仓观望，不操作** |

**如何快速判断？** 看三个数据：
- 昨日涨停家数 vs 跌停家数
- 昨日连板股今日表现（开盘是否高开）
- 昨日涨停今日炸板率

**情绪冰点和退潮期，最正确的操作就是空仓。** 超短高手一年真正出手可能只有80-100天。

---

**2. 确定当日主线题材**

早盘前浏览：
- 隔夜消息面（政策、突发事件、外盘）
- 竞价异动板块
- 昨日龙虎榜数据（哪些席位在买什么方向）

**核心原则：只做主线，不做支线**

| 类型 | 特征 | 能不能做 |
|------|------|---------|
| 主线 | 连续2天以上发酵，有明确龙头，板块内多只涨停 | ✅ |
| 支线 | 今天突然冒出来，只有1-2只涨停 | ❌ 成功率低 |
| 昨日主线 | 昨天是主线但今天资金已转向 | ❌ 接力容易吃面 |

---

**3. 锁定候选标的（不超过5只）**

从主线题材中，按以下标准筛选：

| 优先级 | 条件 |
|--------|------|
| 第一 | **辨识度最高的龙头**（板块内最先涨停、连板最高、市场公认） |
| 第二 | 龙头的补涨（龙头太高时，板块内低位首板） |
| 第三 | 断板后的龙头反包（需要情绪配合） |

**不碰的**：
- 跟风股（板块第3、4、5只涨停的）
- 一字板（买不到，开板就是出货）
- 无量涨停后次日（说明筹码锁定差）

---

### 二、竞价阶段（9:15-9:25）

**这10分钟决定今天的计划是否执行**

**观察重点**：

**1. 竞价量能**
- 候选标的竞价成交量 vs 昨日竞价量
- **放量高开3-5%** = 资金认可，可做
- **缩量高开7%+** = 可能是一字板抢筹，观察
- **低开或平开** = 计划作废，放弃

**2. 竞价分歧度**
- 9:15-9:20 有大单砸但9:20-9:25 回升 → 洗盘后确认，好信号
- 9:15-9:20 拉高但9:20-9:25 持续回落 → 诱多，回避

---

### 三、盘中执行（9:30-10:30）

**核心窗口只有前一小时**，超短线大部分操作在这里完成。

**买点模型（三种）**：

**模型A：打板（涨停板买入）**

适用于：确定性最高的龙头股

| 条件 | 要求 |
|------|------|
| 封板时间 | 10:30前封板最佳 |
| 封单量 | 流通市值的5%以上 |
| 换手率 | 首板5-15%最健康 |
| 分时 | 快速直线拉板 > 震荡磨板 |
| 炸板 | 允许1次炸板回封，2次以上放弃 |

**模型B：低吸龙头分歧转一致**

适用于：连板龙头高开后分歧（回落），然后资金回流

| 时间 | 动作 |
|------|------|
| 9:30-9:45 | 龙头高开后回落，观察不动 |
| 9:45-10:00 | 如果回落不破均线且有大单承接 → 这是低吸点 |
| 10:00后 | 如果持续走弱 → 放弃 |

**模型C：半路板**

分时走势呈现N字形（拉升→回调→再拉升），在第二波启动时买入。

---

**每次只执行一个交易，买入后立刻设定**：

| 项目 | 标准 |
|------|------|
| 止损线 | 买入价-5%（打板则按炸板不回封止损） |
| 目标 | 次日竞价/开盘高点 |
| 最大持有 | 3天不涨停就走 |

---

### 四、盘后复盘（15:00-16:00）

**这是超短线真正的核心功课，比盘中操作更重要**

每天必须记录：

**1. 市场数据**
- 涨停家数/跌停家数/炸板家数
- 连板梯队（最高板是几板，有几只）
- 主线题材演变

**2. 个股复盘**
- 今日所有涨停股逐一浏览（不用深看，扫一眼题材和板块归属）
- 龙虎榜数据（哪些席位在买卖）
- 明日重点候选标的

**3. 自我交易记录**

## 仓位管理铁律

| 规则 | 内容 |
|------|------|
| 单票仓位 | 30-50%（最多1-2只） |
| 总仓位 | 情绪好时满仓，情绪差时空仓 |
| 亏损上限 | 单日亏损>总资金3%，当日停手 |
| 连亏处理 | 连亏3次，强制休息3天 |
| 回撤红线 | 月回撤>10%，停手一周复盘 |
